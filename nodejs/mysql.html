mysql

数据库

用来组织，存储和管理数据的仓库
数据库管理系统，简称，数据库
用户可以对数据库中的数据进行增删改查等操作

常见的数据库分类

  1. MySQL community不收费，enterprise收费
  2. Oracle 收费
  3. SQL Server 收费
  4. Mongodb community不收费，enterprise收费
    前三者属于传统型数据库又叫做关系型数据库或SQL数据库，三者设计理念相同，用法比较类似
  • workbench或navicat或dataGrip都可以连接数据库

SQL

是结构化查询语言，专门用来访问和处理数据库的编程语言，能够让我们以编程的形式，操作数据库里面的数据

  • 关键点
  1. SQL是数据库编程语言
  2. 使用SQL写出来的代码，叫做SQL语句
  3. SQL语言只能在关系型数据库中使用
  • 如何使用SQL从数据表中
  1. 查询数据 select
  • SQL语句对大小写不敏感，SELECT等效于select，FROM等效于from
  1. 插入数据 insert into
  1. 更新数据 update
    修改表中的数据
  1. 删除数据 delete
    删除表中的行
    DELETE FROM 表名称 WHERE 列名称=值
  • 额外需要掌握
  1. where 条件
    where句子用于限定选择的标准，在select、update、delete语句中，皆可使用where子句来限定选择的标准
  • 运算符有两个特别的
    between 在某个范围内
    like 搜索某种模式
  1. and 和 or运算符
    在where语句中把两个或多个条件结合起来
  • and表示必须同时满足多个条件，相当于JS中的 &&
  • or表示只要满足任意一个条件即可，相当于JS中的 ||
  1. order by 排序
    ORDER BY 语句用于根据指定的列对结果集进行排序
    ORDER BY 语句默认按照升序对记录进行排序
    ASC关键字代表升序排序

如果希望按照降序对记录进行排序，可以使用DESC关键字
SELECT * FROM users ORDER BY id DESC

  • 多重排序
  1. count() 函数
    用于返回查询结果的总数据条数
    SELECT COUNT(    ) FROM 表名称
  • 如果希望给查询出来的列名称设置别名，可以使用as关键字
  • id记得勾选not null 和 AutoIncrement INT类型
