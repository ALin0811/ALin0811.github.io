什么是TypeScript

  • 以JavaScript为基础构建的语言
  • 一个JavaScript的超集：对js的一个扩展，支持js，在此基础上增添了许多功能。最主要的引入了类型，从动态类型语言变成静态类型语言
  • 可以在任何支持JS的平台中执行
  • TS不能被JS解析器直接执行

  1. 类型
  2. 支持ES的新特性
  3. 添加ES不具备的新特性

搭建TypeScript项目

  1. tsc –init 生成tsconfig.json
  2. npm add @types/node -D 配置环境，node 的核心包具备类型提示也就是package.json
  3. tsc module.ts 对文件完成编译 生成一个module.js
  4. npm add ts-node-dev -D
  5. tsc module.ts -w 直接监听变化，无需ts写什么，每一次都执行tsc module.ts，module.js才能跟着变化
  6. 但如果有特别多的ts文件，总不能每一个都开启一个监听 直接tsc回车都编译完成
  7. 进行ts.config.js配置
  8. 下载webpack相关进行配置
  9. 编译webpack打包
  10. npm i less less-loader css-loader style-loader 下载样式
  11. 有一些旧版本的样式，用postcss npm i -D postcss postcss-loader postcss-preset-env

[Tip]

  1. js代码可以不加分号，除了小括号、中括号开头的代码
  2. ts以括号开头的语句，必须要加分号结尾，或者在下一行的开头加上分号也行
